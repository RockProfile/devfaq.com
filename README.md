# Devfaq.com

This repository holds the source code for the devfaq.com site.

## Plugins

### Article

Requires:

* Title
* Text
* Author
* Publish Date

### Calendar

Requires:

* Link to plugin with data published for that day

### FAQ

Requires:

* Question
* Answer

### Featured Function

Requires:

* Name
* Documentation
* Description
* Publish Date

### Featured Package

Requires:

* Package Name
* Package download
* Documentation
* Description
* Publish Date

### News

Requires:

* Title
* Text
* Publish Date

### Homepage

Show the data from the plugin for the current day. If no
content exists show the last content
